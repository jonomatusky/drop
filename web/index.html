Demo: how drop works

This page is the demo. It's a single HTML file that became a clean public URL on the open internet by running one command — no build step, no server to run, nothing to sign up for. 

$ drop web/index.html --public --slug demo
  ↳ drop.jonomatus.ky/demo/

That's it. The page you're reading is the result.

drop is a personal, self-hosted publishing tool built by jonomatusky and run on his own Cloudflare account. It's open source — the full writeup of how it works lives in the README: github.com/jonomatusky/drop.
